# SmartBuy AI

SmartBuy AI is a Multi-Agent Shopping Intelligence System built using Python, Gemini, LangGraph, and Web Scraping.

## Features

* Amazon Product Analysis
* Review Summarization
* Aspect-Based Review Analysis
* AI-Powered Buy Recommendation
* Multi-Agent Architecture
* Caching and Fallback Mechanisms

## Tech Stack

* Python
* LangGraph
* Google Gemini
* BeautifulSoup
* FastAPI
* React (Upcoming)

## Architecture

Product URL
→ Product Agent
→ Review Agent
→ Aspect Agent
→ Verdict Agent

## Future Enhancements

* Aspect Discovery Agent
* Payment Optimization Agent
* Product Recommendation Agent
* RAG-based Review Retrieval
* Multi-Platform Product Comparison
